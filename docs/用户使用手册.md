## 产品简介

**SoC Analyzer Pro** 是一款专业的手机SoC芯片性能深度分析工具，由Android手机端应用和PC端展示平台组成。通过有线(USB/ADB)或无线(WiFi/WebSocket)方式，实现手机芯片性能数据的实时采集、传输和可视化分析。

### 核心功能

- **芯片信息采集**: 读取CPU/GPU/NPU/内存等硬件信息
- **性能跑分测试**: CPU单核/多核、GPU、AI性能跑分
- **实时监控**: CPU/GPU频率、温度、功耗实时曲线
- **压力测试**: 15分钟持续负载，检测性能衰减和温度墙
- **游戏帧率测试**: 帧率曲线、1% Low、卡顿检测
- **专业报告**: 自动生成PDF/HTML测试报告

## 安装指南

### Android手机端安装

1. 下载 `SoC-Analyzer-v1.0.0.apk` 安装包
2. 在手机设置中允许"安装未知来源应用"
3. 点击APK文件进行安装
4. 安装完成后授予必要权限

### PC端安装

#### Windows

1. 下载 `SoC-Analyzer-Pro-1.0.0-x64.msi` 安装程序
2. 双击运行安装程序
3. 按照向导完成安装
4. 从开始菜单启动应用

#### macOS

1. 下载 `SoC-Analyzer-Pro-1.0.0.dmg` 镜像文件
2. 双击打开DMG文件
3. 将应用拖入Applications文件夹
4. 首次运行需在"系统偏好设置-安全性与隐私"中允许运行

## 快速开始

### 步骤1: 启动PC端

1. 打开 SoC Analyzer Pro 桌面应用
2. 应用会自动启动WebSocket服务器
3. 记录显示的IP地址和端口号

### 步骤2: 连接手机

#### 无线连接 (推荐)

1. 确保手机和电脑在同一WiFi网络
2. 打开手机端应用，进入"连接设置"
3. 点击"扫描二维码"或手动输入PC端IP地址
4. 连接成功后状态显示为绑色"已连接"

#### 有线连接 (ADB)

1. 使用USB数据线连接手机和电脑
2. 手机开启"开发者选项"和"USB调试"
3. PC端点击"扫描ADB设备"
4. 选择设备并点击"连接"

### 步骤3: 开始测试

1. 在PC端选择测试项目:
   - **快速跑分**: 约5分钟，测试CPU/GPU/AI性能
   - **完整跑分**: 约15分钟，包含所有测试项目
   - **压力测试**: 15分钟持续负载测试
   - **游戏测试**: 指定应用帧率监控

2. 手机端会同步开始执行测试
3. 测试数据实时传输到PC端显示

## 功能详解

### 1. 芯片信息

显示手机SoC的详细硬件信息:

- **CPU信息**: 型号、核心配置(大中小核)、架构、最大频率
- **GPU信息**: 型号、最大频率
- **NPU信息**: AI加速器型号、算力(TOPS)
- **内存信息**: 类型(LPDDR5X)、容量、带宽
- **制程工艺**: 4nm/5nm等

### 2. 实时监控

实时显示设备运行状态:

| 监控项 | 说明 |
|--------|------|
| CPU频率 | 各核心当前运行频率曲线 |
| CPU使用率 | 总体及各核心使用率 |
| GPU频率 | 当前GPU运行频率 |
| 温度 | CPU/GPU/电池温度曲线 |
| 功耗 | 估算实时功耗(mW) |
| 内存 | 已用/总容量 |

### 3. 性能跑分

#### CPU跑分
- **单核分数**: 单核心整数+浮点运算性能
- **多核分数**: 多核心并行计算性能
- **内存带宽**: 内存读写速度

#### GPU跑分
- **填充率测试**: 像素填充性能
- **纹理采样测试**: 纹理处理性能
- **计算着色器测试**: GPU通用计算性能

#### AI跑分
- **NNAPI测试**: Android神经网络API性能
- **INT8推理**: 量化模型推理速度
- **FP16推理**: 半精度浮点推理速度

### 4. 压力测试

持续15分钟的CPU满载测试，检测:

- **性能衰减**: 对比初始和最终性能
- **温度墙**: 记录触发降频的温度阈值
- **降频时间点**: 标注开始降频的时间
- **稳定性评分**: 综合评估散热和性能稳定性

### 5. 游戏帧率测试

监控指定应用的帧率表现:

- **平均帧率**: 测试期间的平均FPS
- **1% Low**: 最差1%帧的平均帧率
- **0.1% Low**: 最差0.1%帧的平均帧率
- **卡顿次数**: 检测到的Jank帧数量
- **帧时间分布**: 每帧渲染时间直方图

### 6. 报告生成

支持导出测试报告:

- **PDF报告**: 专业排版，适合打印和分享
- **HTML报告**: 网页格式，可在线查看
- **数据导出**: JSON/CSV格式原始数据

## 常见问题

### Q1: 无法连接到手机

**解决方案**:
1. 确认手机和电脑在同一WiFi网络
2. 检查手机防火墙是否阻止了连接
3. 尝试使用USB有线连接
4. 重启手机端和PC端应用

### Q2: 有线连接检测不到设备

**解决方案**:
1. 确认已开启"开发者选项"和"USB调试"
2. 在手机上点击"允许USB调试"
3. 更换USB数据线(部分线缆仅支持充电)
4. 安装设备对应的USB驱动程序

### Q3: 测试过程中手机发热严重

**说明**: 跑分和压力测试会让CPU/GPU满载运行，发热属于正常现象。
**建议**: 
- 测试前关闭不必要的后台应用
- 在室温环境下进行测试
- 取下手机保护壳以改善散热

### Q4: 跑分结果偏低

**可能原因**:
- 手机电量低于20%，系统限制了性能
- 后台有占用资源的应用
- 设备温度过高导致降频
- 省电模式开启

**解决方案**: 充满电、关闭省电模式、清理后台应用后重新测试

### Q5: 游戏帧率测试不准确

**解决方案**:
1. 确保测试期间没有通知打扰
2. 关闭屏幕亮度自动调节
3. 固定屏幕刷新率(如有此选项)
4. 测试时长建议超过3分钟
